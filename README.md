AI 解决方案专家 | 个人作品集 (Portfolio)

🌟 项目简介

这是一个静态网页作品集，旨在展示一位集生物医学背景、市场实战经验和AI 解决方案开发能力于一身的专家。

本作品集的核心目标是清晰地阐述“将复杂的行业痛点转化为可落地的 AI 解决方案”的方法论，并辅以具体的垂直行业（医疗/新零售）洞察和实战项目案例。

🚀 核心亮点与价值

作品集主要分为以下几个部分，以建立完整的个人品牌：

个人定位 (Hero Section): 突出“生物医学硕士 x 市场实战 x AI 开发者”的独特复合背景。

方法论 (Insights): 详细展示了 AI 解决方案的闭环落地框架（从场景诊断到人机协作的 4 个步骤）。

行业认知 (Industry Matrix): 深度剖析了医疗/生命科学和新零售/时尚电商两大垂直领域的痛点、AI 解决方案及具体应用案例（如 Google DeepMind、推想科技、PathAI、Insilico Medicine 等）。

实战案例 (Projects): 突出两个关键项目：

AI 智能穿搭小助手： 使用 RAG、多模态 Agent 和 LangChain 进行从 0 到 1 的独立开发实践。

医疗行业销售培训： 侧重知识图谱 (Knowledge Graph) 和数据结构化在解决知识门槛高、培训周期长等行业痛点中的应用。

数字分身 (Avatar Chat): 一个Lite-Agent交互占位符，展示未来可集成的、基于个人知识库构建的问答系统。

🛠️ 技术栈

本项目是一个轻量级、响应式、零依赖（运行时）的静态网页，适合快速部署和高效展示。
